# Sample videos

Drop short aquarium clips here to demo the dashboard without a webcam.

```bash
bash scripts/run_local.sh --video samples/aquarium.mp4
```

The dashboard will loop the file forever, so a 30-second clip is plenty.

The repo doesn't bundle a clip out of the box (binary blobs bloat the
repo). A few free sources:

- Pixabay: <https://pixabay.com/videos/search/aquarium/>  (CC0)
- Pexels: <https://www.pexels.com/search/videos/aquarium/>  (free for personal/commercial use)

Save your downloaded clip as `samples/aquarium.mp4` and you're set.
